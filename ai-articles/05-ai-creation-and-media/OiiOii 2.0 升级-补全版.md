# OiiOii升级2.0，我终于认真做了一次 AI 视频

> 日期：2026-06-15


玩 AI 这么久了，我一直没怎么认真折腾过视频。

不是我不想做，是每次一想到做视频，我头就开始大。

写脚本、文生图、图生视频、修改、剪辑、配音。。。。。。

这些东西至少还有一个比较清楚的工作流。你知道自己要给什么材料，要怎么验证，要怎么改才行。

但 AI 视频不一样，注意我说的是 **AI 视频。**

我之前看很多大号和创作者都在推荐 OiiOii，说它更适合做 ACG、短剧、动画、自媒体视频这些内容。

官网在这：**https://www.oiioii.ai/**

刚好这次 OiiOii 2.0 升级，我就拿它认真试了一下。

![image-20260614200738133](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614200738133.png)

先说结论。

它把视频创作这件事，拆成了一个更接近真实创作团队的流程，这是最吸引我的一点。

当你对着一个输入框输入自己的想法的时候，你背后不只是一个 Agent。

它是有艺术总监、编剧、角色设计师、场景设计师、分镜师、音效总监这些 Agent，一步一步把一个想法往成片推。

OiiOii 2.0 这次比较核心的升级有三个：

- **拉片复刻**：上传视频后自动分析镜头语言，最多从 18 个专业维度拆解视频，再替换主体、角色、场景，做出新的版本。
- **智能画布**：Agent 能理解整个画布，你可以在画布里针对角色、场景、分镜、音乐等局部直接修改。
- **Skill 库**：内置很多行业创作工具包，覆盖自媒体、广告营销、周边设计等场景。

![image-20260614084007233](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614084007233.png)

我认真体验了一下 OiiOii ，做了两个视频。

第一个是先用智能画布从 0 做一次视频。

第二个再用拉片复刻，把一个世界杯名场面做成 AI 圈二创。

### 第一步，先让 OiiOii 拆镜头

我把 C 罗这段视频传上去之后，OiiOii 会先做拉片分析。

它会把视频拆成镜头、时间、画面内容、角色动作、镜头语言、视觉氛围、声音节奏这些维度。

![image-20260614225657294](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614225657294.png)

这一步对我这种视频新手来说挺有用。

因为我以前想复刻一个视频，只能靠自己反复看。

这个镜头几秒？

这个镜头是特写还是中景？

人物在画面里是正面还是背面？

观众席什么时候切？

慢动作在哪里？

这些东西如果不想清楚，后面基本就是凭感觉写 Prompt。

OiiOii 的拉片复刻，相当于先帮你做了一份镜头分析稿。

这一步解决的是“我知道这个视频好看，但我不知道它到底怎么拍出来”的问题。

但如果你能把镜头结构拆出来，再换成自己的内容，就更像创作。

## 智能画布

这个功能我一开始以为就是普通自由画布。

类似把素材、角色、场景、分镜都放在一个大白板上，然后你自己拖来拖去。

但用了一下发现，它和自由画布还是有区别的。

自由画布更像一个容器。

它负责放东西，你把图片放进去，把文字放进去，把分镜放进去，然后自己判断下一步该怎么改。

**OiiOii 的智能画布更像一个能看懂上下文的工作台。**

![image-20260614232719138](https://cdn.jsdelivr.net/gh/doggaifan/picbed@main/image-20260614232719138.png)

它不是只知道你输入框里刚说了什么。

它能结合当前画布里的角色、场景、分镜、音乐这些内容，再调用对应的 Agent 去改某一个板块。

比如角色不稳定，那就去改角色设定。

场景不对，就去找场景设计师改布景。

分镜节奏不对，就让分镜师重新调整镜头。

音乐不对，就让音效总监改声音氛围。

它不像一个只能“重新生成”的黑盒。

更像一个可以局部返工的工作台。

我这次从 0 做视频的时候，整个流程是这样走的。

首先出场的是艺术总监。

它负责根据你的创意，先搭起整个故事框架。

它会问你视频的基础参数，比如多长时间、横版还是竖版、中文还是日文、整体风格偏写实还是动画。

![image-20260614151433279](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614151433279.png)

角色设计师会根据剧本设定人物形象。

![image-20260614230855200](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614230855200.png)

场景设计师会继续为你搭建布景。

![image-20260614230913837](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614230913837.png)

然后是分镜。

同一个故事，分镜不同，出来的感觉完全不一样。

你是要平铺直叙，还是先给特写。

你是要慢慢铺情绪，还是节奏紧一点。

你是要人物先出场，还是先交代场景。

分镜师理解完需求之后，会为你设计分镜场景。

![image-20260614155225126](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614155225126.png)

再到音乐。

视频这东西，声音很容易被忽略。

但很多时候，声音的重要性至少占一半，尤其是短视频。

前几秒观众能不能被拉住，除了画面，节奏和音乐也很关键。

这些板块全部都可以继续修改。

**所以我觉得智能画布真正有价值的，是你可以在具体环节里调用具体 Agent 来修改对应模块的内容。**

角色不对，不用整条重来。

场景不对，不用整条重来。

分镜不对，也不用整条重来。

这对 AI 视频很重要。

整体用下来，我觉得智能画布最像是把“改视频”这件事变得更可控。

它不是让你每次都回到输入框，从头再写一遍。

而是让你能在画布里找到具体问题，然后调用对应 Agent 去改。

这个是它和普通自由画布最大的区别。

## OiiOii 的 Skill 库

除了拉片复刻和智能画布之外，还有一个 Skill 库。

![image-20260614200946644](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260614200946644.png)

这个东西可以理解成 OiiOii 内置的一组创作工具包。

里面覆盖了自媒体创作、广告营销、周边设计等不同方向。

如果你已经很懂视频创作，可能会觉得 Skill 库只是省一点配置时间。

但如果你是刚开始做 AI 视频的人，这个东西还是有用的，节省你大量的选题时间。

## 我觉得它真正降低的是创作门槛

我以前对 AI 视频工具有个偏见。

我觉得它们更像是“片段生成器”。

你输入一段话，它给你吐一段视频。

如果运气好，这段视频能用。

如果运气不好，那就只能浪费 token 继续重新搞。

整个过程像是在抽盲盒，prompt 只能提高准确率，但是上限很低。

但 OiiOii 这次给我的感觉不是这样。

它更像是把视频制作拆成了一条工作流。

先有创意，然后有剧本 -> 角色 -> 场景 -> 分镜 -> 音乐 -> 成片。

智能画布是从 0 到 1。

它帮你把一个模糊创意拆成可执行流程。

拉片复刻是从 1 到 N。

它帮你从一个成熟视频里学习镜头结构，再换成自己的角色和故事。

这两个能力合在一起，对三类人比较有用。

第一类是动画自媒体。

你不一定有完整团队，但你需要稳定产出角色、场景、分镜和短片。

第二类是游戏宣发和短剧团队。

你们可能有 IP 或人物设定，但缺的是快速把设定变成短视频素材。

第三类是想追热点的普通创作者。

比如世界杯来了，你想做一个相关视频。

以前可能只能混剪、配音、写段子。

现在你可以拿一个热门视频的结构，改成自己的故事。

这就比单纯“蹭热点”多了一层创作层面。

## 我的结论

这次试完 OiiOii升级2.0，我最大的感受是：

AI 视频终于开始有稳定的工作流了。

很多过去需要专业团队才能完成的事情，现在可能会被更小的团队、更快地做出来。

就像《双城记》里说的，这是最好的时代，也是最坏的时代。
