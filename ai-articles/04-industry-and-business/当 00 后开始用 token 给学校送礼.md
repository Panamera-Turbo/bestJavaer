# 当 00 后开始用 token 给学校送礼

> 日期：2026-05-18

看到有意思的新闻：三个 00 后给母校"捐了 20 亿"，初以为是 20 亿，点进去一看才知道，是 20 亿 token。

这条新闻的全部戏剧性，正恰恰就是 token 这两个字。

![image-20260518152307594](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260518152307594.png)

"亿"曾经是个有重量的单位。捐一个亿是新闻，捐十个亿是大新闻，捐二十个亿足够让母校官网首屏给你挂一个学期。

20 亿 token 在 2026 年是什么概念？

国产模型对命中缓存的输入定价非常激进。一个跑得勤的研究生，一个月烧几亿 token 不稀奇。20 亿听起来宏大，分到一个实验室里，差不多就是一两个学期本科生小组项目的预算。

网友给的数字更狠，说走 deepseek v4 flash 的缓存，大概**一百块上下**。这个估值未必精准，量级是对的——和"亿"这个字制造的体感，差了七个数量级。

---

## token 进入了礼物经济中

十年前没人会发新闻说"我向母校捐赠 200 GB 带宽"，因为带宽从未在大众语义里成为礼物。token 这次不一样。

00 后是和模型一起长大的一批。token 对他们来说就是日常资源，跟水电、奶茶一个层次。送 token 给学弟学妹，送 token 给老师做研究，这很正常。

学校愿意接受，也是一个微妙的信号。它说明 token 已经普及到双方都听得懂，又还没便宜到"送这个有点搞笑"的程度，而且刚好卡在两者之间。

> 礼物的重量从来不靠钱撑起来，靠的是双方对那件东西有共识。

本来其实是双方都达成共识的一件事情。

但是拉出来放到网上让大家看到之后。

事情就变了。

以至于这件事情，有点戏谑。

因为只要把 AI 重度融入日常工作之后的不管是程序员、产品、设计也好。

每天消耗个千八百亿 token ，都很正常。

回到最开始那个戏谑感。

这个事情显示在另一个纬度下，它在另一个不容易察觉的地方。

这条新闻完整地展示了一个正在发生的锚点转换的过程。

"亿"这个字曾经能独立携带价值判断，现在你得先问一句"亿的是什么"。

很多看起来稳固的衡量单位，都在悄悄进入这种"补不出单位读不懂"的状态。token 是一个，算力是一个，注意力也是一个。再过几年，"我给你转了一百万 X"这类句式会越来越常见，X 是什么，决定了这句话是慷慨、是玩笑、还是侮辱。

二十亿 token 这条新闻可能火一会儿就完了。

但是背后的锚点转换，从这件事情上算是开了一个头。

让你看见货币、价值、礼物这几个老词，正在被新的单位悄悄重新定义。
