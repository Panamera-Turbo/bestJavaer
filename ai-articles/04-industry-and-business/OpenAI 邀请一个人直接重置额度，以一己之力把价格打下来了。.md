# OpenAI 邀请一个人直接重置额度，以一己之力把价格打下来了。

> 日期：2026-06-12


前段时间，就传出来 OpenAI 上线了一个操作，这个操作不是什么插件，功能甚至重大更新。

但却凭借这个操作得到广大用户的认可和好评。

以至于，这个操作被传出来是从拼多多学过去的，甚至有人说 Codex 招到了拼多多的增长经理。

![image-20260612105906073](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260612105906073.png)

那就是，OpenAI 可以通过邀请一个人的方式，直接给你把 Plus/Pro 额度直接重置。

<img src="https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260612084509726.png" alt="image-20260612084509726" style="zoom:50%;" />

未来`两周`内，Plus 和 Pro 用户可以邀请最多三位朋友试用 Codex。 

当一位朋友发送他们的第一条 Codex 消息时，你们双方都会获得额度重置。

<img src="https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260612085700379.png" alt="image-20260612085700379" style="zoom:50%;" />

目前已经全量上线了，你可以在 codex 设置里面看到 `invite a friend` 入口。

![image-20260612085914146](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260612085914146.png)

这个操作一出来，可以代表着你未来两周内，可以不限额度进行拉满夯了。

在 OpenAI 眼里，用户量要比额度重要的多得多。

Anthropic 那边，Claude Code 借着 fable 5 的声势，把限时体验的窗口直接开到 6 月 22 号，意图再明显不过，在这短短十几天里，让尽可能多的码农上手 Claude，把自己焊进他们的日常工作流里。

而 OpenAI 的回应，不是降价，不是加功能，而是直接给老用户发福利。

你每拉一个朋友入坑 Codex，我们就给你重置额度，让你接着奏乐接着舞。

不过你仔细品品这个机制，你会发现它几乎就是拼多多“砍一刀”的 AI 版本。

![image-20260612093757812](https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260612093757812.png)

但是，OpenAI 学的还不够彻底。

拼多多的机制是，你不知道砍成某个商品需要拉多少人，你得持续不断的拉人才行，只要你拉不够人，那么不管你拉多少用户都没用，这就让大家陷入到拉新焦虑中。

OpenAI 还是太有良心了，只要你拉一个朋友发第一条 Codex 消息，你俩的 Plus/Pro 额度直接满血复活。

所以，“从拼多多学过去的”这个说法，听起来像是个段子，但细想其实是一句“极高”的评价。

这意味着，当最头部的 AI 公司都开始琢磨社交裂变的时候，AI 产品的竞争，已经彻底进入了拼增量、抢地盘的阶段。

---

接下来两周，看点还是很多。

**第一个看点是，这种机制会不会被玩坏了，毕竟 OpenAI 已经被玩坏了太多次了。**

不过我看已经有人已经开始琢磨用虚拟号码、临时邮箱批量制造“新朋友”，自己拉自己刷重置。

以 OpenAI 的风控能力，后续大概率会加入设备指纹、行为检测之类的限制，但在规则窗口期，免不了一波薅羊毛的狂欢。

但 OpenAI 可能并不在意，因为即便是“假朋友”，也会产生真实的使用数据，这些数据对模型训练和产品迭代本身就有价值。

**第二个看点是，Anthropic 会怎么接招。**

Claude Code fable 5 的限时到 22 号，OpenAI 的邀请窗口也是两周，时间几乎是咬在一起的。两边都在制造紧迫感，都在试图把开发者圈进自己的生态。

如果 Claude 那边接下来也推出类似的“邀请解锁额度”或者“组队得 token”活动，千万别觉得奇怪。

当行业头牌都开始拼多多化，就别指望谁还能端着架子了，毕竟你还得靠用户给你付费。

不过以 A 社的尿性来看，我觉得大概率不会。

**第三个看点，更是长期的影响：额度正在入侵用户的心智。**

过去大家比的是模型跑分怎么样、模型能力强不强、回答好不好；

现在，还多了一层“我在这里有人脉，有额度可以分享”的社交资产。

这种社交换算力的心智植入，一旦成型，迁移成本会变得极高。

所以，不管你怎么想，双方打起来，这对广大用户来说是一件大好事儿。

至于这股拼多多风接下来会刮到什么程度，也许不久的将来，我们就会看到“签到领 token”，“邀请好友组队抽 GPT-5.6 的内测资格”，“连续分享七天解锁 GPT-image-2 图片的不限量”这样的操作，真到那一天，也不用太惊讶。

毕竟，在用户增长面前，额度，真的就只是拿来送的。
